# Интернет магазин 

## Описание проекта
Проект разрабатывается в рамках изучения django и функционал все время растёт.
На данном этапе приложение позволяет следующее:
- Открывать страницы "Каталог" и "Контакты" с подключенными стилями bootstrap.
- На странице контакты имеется форма обратной связи которая передает сообщения в консоль сервера.

## Структура проекта:
### Корень проекта содержит:
    - README.md - содергит описание проекта.

### Стандартный проект Django:
    - Содержит, кроме стандартного набора, папку templates с шаблонами страниц "Каталог" и "Контакты".
    - так же в проекте локально, в папке static, расположены файйл стилей и java-скрипт от bootstrap.

## Установка

1. Клонируйте репозиторий
'''
git clone https://github.com/alekseyzvyagintsev/middle_web_app.git
'''
2. Установите зависимости
'''
pip install -r requirements.txt
или
poetry install
'''
3. Активируйте виртуальное окружение
"source .venv/bin/activate"
4. Запустите локальный сервер
"python3 manage.py runserver   или   python manage.py runserver"
5. Откройте в браузере локальный адрес
"http://127.0.0.1:8000/"

## Использование
В настоящий момент использовать функционал возможно следующие возможности:
открывать страницы по адресам:
- http://127.0.0.1:8000/ домашняя страница "Каталог"
- http://127.0.0.1:8000/contact/ "Контакты"

На странице "Контакты" работает форма обратной связи. Функционал описан выше.
